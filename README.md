# рҹҸ—пёҸ Navngivningsprincipper

Alle repositories skal fГёlge denne struktur:

Hvor:

- **kategori**: overordnet fagomrГҘde  
- **underemne** (valgfrit): teknologi, sprog eller platform  
- **projekt**: titel eller formГҘl  

Der bruges **smГҘ bogstaver**, **engelske ord** og **bindestreger** (`-`).  
Specialtegn, mellemrum og `/` mГҘ **ikke** bruges.
